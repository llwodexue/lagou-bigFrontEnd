## 超过 Number 最大值怎么处理

JS 超过 Number 最大值的数怎么处理？`Number.MAX_VALUE`

背景：

1. 大数据计算
2. 格式展示
3. 用户输入

解决方案：

1. BigInt 来处理大数据
2. decimal.js、big.js
3. 格式化，格式化用户好读的格式 100,000,000 -> 1亿
4. 表单校验，不允许用户输入超过多少位的数字
