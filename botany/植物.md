## 小麦

### 麦子

野生的麦子既开不出明显的花，也结不出什么像样的果实，它们的种子被包裹在一种叫做"颖果"的坚硬外皮中，其上还长着又尖又长的麦芒。不但如此，为了快速繁殖，野麦种子一旦成熟，穗子的轴很快就会碎掉，麦籽直接掉一地

- 颖果：禾本科特有的果实类型，果皮与种皮高度愈合难以分离

![image-20250224145108131](https://gitee.com/lilyn/pic/raw/master/md-img/image-20250224145108131.png)

在东汉时期开始，来自西方的圆形石磨开始在中国民间普及了开来，从此生产队的驴终于有了磨盘可以拉了，小麦也成了香饽饽。磨盘的推广在之后把中国本土产的大豆和稻米变成了豆腐和米粉

小麦为中国人提供了 40% 的口粮，略低于水稻。而在世界范围内，小麦则是毋庸置疑的第一主粮。小麦相比于其他主粮谷物而言有一个明显特点：它含有特别高的蛋白质。这不但让小麦成为了营养全面的主粮，也赋予了面粉如橡皮泥一般可以随意塑性的特点

### 啤酒

#### 啤酒的四大原料

啤酒被称为"液体面包"，其核心原料只有四种：水、麦芽、酒花、酵母

| 原料 | 作用 | 说明 |
| ---- | ---- | ---- |
| 水 | 载体 | 占总重 90% 以上，水质直接影响啤酒风格（软水适合淡色艾尔，硬水适合世涛） |
| 麦芽 | 糖分来源 | 提供可发酵糖和麦芽风味，大麦最常用，小麦次之 |
| 酒花 | 苦味与香气 | 提供苦味平衡麦芽甜度，同时带来花香、果香、松香等 |
| 酵母 | 发酵引擎 | 将糖转化为酒精和 CO₂，同时产生大量风味副产物 |

#### 啤酒的酿造流程

```
制麦 → 粉碎 → 糖化 → 过滤 → 煮沸(加酒花) → 冷却 → 发酵 → 熟成 → 灌装
```

1. **制麦（Malting）**：将大麦浸泡、发芽、烘干。发芽过程中种子产生酶，这些酶在后续糖化阶段负责将淀粉分解为可发酵糖
2. **粉碎（Milling）**：将麦芽粉碎，破坏淀粉颗粒外层，便于糖化
3. **糖化（Mashing）**：将粉碎的麦芽与热水混合，在 62-72°C 下保温。酶将淀粉分解为麦芽糖、麦芽三糖等可发酵糖
4. **过滤（Lautering）**：将麦汁与麦糟分离，得到澄清的糖化液
5. **煮沸（Boiling）**：煮沸麦汁 60-90 分钟，期间分批加入酒花。早加酒花提供苦味，晚加酒花提供香气
6. **冷却（Cooling）**：快速降温至发酵温度，避免杂菌污染
7. **发酵（Fermentation）**：加入酵母，主发酵通常持续 5-14 天
8. **熟成（Conditioning）**：低温静置，让风味融合、杂质沉淀
9. **灌装（Packaging）**：瓶装、罐装或桶装

#### 啤酒的分类

啤酒按酵母发酵方式可分为两大类：

| 分类 | 酵母类型 | 发酵温度 | 发酵位置 | 特点 | 代表风格 |
| ---- | -------- | -------- | -------- | ---- | -------- |
| 艾尔（Ale） | 上发酵酵母 | 15-25°C | 发酵液上层 | 风味丰富，果香突出 | IPA、世涛、小麦啤、比利时艾尔 |
| 拉格（Lager） | 下发酵酵母 | 7-13°C | 发酵液底部 | 口感干净、清爽 | 皮尔森、博克、德式小麦拉格 |

> 全球约 85% 的啤酒是拉格，因为拉格生产工艺更稳定、更适合大规模工业化生产

#### 小麦与大麦在啤酒中的对比

传统的啤酒一般用大麦麦芽酿造，过去小麦主要是用于主粮的作物，很少用于酿造。不过在啤酒圣地德国巴伐利亚，最受欢迎的却是小麦啤酒，它主要用小麦麦芽酿造，其使用量不低于 50%。因其酒体浑浊，通常也被称为白啤

| 特性 | 小麦 | 大麦 |
| ---- | ---- | ---- |
| 淀粉含量 | 淀粉颗粒细小，易糊化，糖化效率高 | 淀粉颗粒较大，需更长时间糖化 |
| 蛋白质含量 | 蛋白质含量较高（12%-15%），赋予泡沫细腻性 | 蛋白质较低（8%-10%），泡沫较持久但稍粗糙 |
| β-葡聚糖 | 含量高，增加酒体粘稠度和浑浊感 | 含量较低，酒体更清澈 |
| 麸皮含量 | 麸皮少，成品更顺滑 | 麸皮多，可能增加苦味和粗糙感 |

#### 小麦啤酒（Wheat Beer）

- **主原料**：小麦芽占比 ≥50%
- **糖化方式**：
  - 低温糖化：降低酶活性，保留更多果香前体物质（适合比利时风格）
  - 高温糖化：快速提取麦汁，风味更浓郁（如德国小麦啤酒）
- **发酵特点**：
  - 多用上发酵酵母（Ale Yeast），发酵活跃，产生酯类（香蕉、丁香）和酚类（辛香）
- **口感特征**：
  - 酒体浑浊（酵母和蛋白质悬浮），颜色呈淡黄至金黄色
  - 泡沫丰富细腻，口感顺滑
  - 典型风味：香蕉、丁香、柑橘、面包

#### 大麦啤酒（Barley Beer）

- **主原料**：大麦芽占比 ≥90%
- **糖化方式**：
  - 阶梯糖化：分阶段升温，充分分解淀粉为麦芽糖（适合复杂麦芽风味）
  - 单一糖化：快速完成，保留更多 β-葡聚糖（增加粘稠感）
- **发酵特点**：
  - 拉格用下发酵酵母（Lager Yeast），低温发酵（4-10°C），口感干净纯粹
  - 艾尔用上发酵酵母，高温发酵（15-25°C），风味更丰富
- **口感特征**：
  - 酒体清澈（经过过滤），颜色从淡黄到深黑不等
  - 风味跨度极大，从清爽淡啤到浓郁世涛

#### 其他经典啤酒风格

##### 皮尔森（Pilsner）

- 起源于捷克皮尔森城（1842 年），是世界上第一款金黄色拉格啤酒
- 特点：清澈金黄、苦度适中、酒花香气明显、口感清爽
- 代表：Pilsner Urquell（皮尔森凯旋）、Staropramen（老磨坊）

##### IPA（印度淡色艾尔）

- 起源于英国，最初为出口印度而大量添加酒花防腐
- 特点：酒花香气浓郁（热带水果、松针、柑橘），苦度高
- 子风格：
  - **英式 IPA**：苦度适中，麦芽底色明显
  - **美式 IPA**：酒花香气奔放，苦度高
  - **双 IPA（DIPA）**：酒花加倍，苦度和香气更强
  - **新英格兰 IPA（NEIPA）**：酒体浑浊，果汁感强，苦度低
- 代表：Stone IPA、Sierra Nevada、Tree House

##### 世涛（Stout）

- 使用深度烘焙大麦甚至黑麦芽，颜色深黑
- 特点：咖啡、巧克力、焦糖风味，口感厚重
- 子风格：
  - **干世涛（Dry Stout）**：清爽不甜，代表 Guinness（健力士）
  - **牛奶世涛（Milk Stout）**：添加乳糖，甜感明显
  - **帝国世涛（Imperial Stout）**：酒精度高达 8%-12%，适合陈年
  - **燕麦世涛（Oatmeal Stout）**：添加燕麦，口感丝滑

##### 博克（Bock）

- 德国强酿拉格，酒精度 6.3%-7.2% ABV
- 特点：麦芽风味浓郁，焦糖、太妃糖香气
- 子风格：双博克（Doppelbock，酒精度 7%-12%）、冰博克（Eisbock，冷冻浓缩，酒精度 10%+）

##### 兰比克（Lambic）

- 比利时特有，使用自然野生酵母和细菌发酵
- 特点：酸味突出，马厩、湿草等独特风味
- **Gueuze（贵兹）**：将不同年份的兰比克混合，产生复杂气泡和酸味层次
- **Fruit Lambic**：加入樱桃（Kriek）、树莓（Framboise）等水果

#### 啤酒品鉴指南

##### 观色

| 颜色 | 可能风格 | 说明 |
| ---- | -------- | ---- |
| 淡黄色 | 皮尔森、淡色拉格 | 使用浅色麦芽，清澈透亮 |
| 金黄色浑浊 | 德式小麦啤 | 酵母和蛋白质悬浮造成浑浊 |
| 琥珀色 | 琥珀艾尔、棕色艾尔 | 使用焦糖麦芽 |
| 深棕色 | 世涛、波特 | 使用烘焙麦芽 |
| 深黑色 | 帝国世涛、俄罗斯世涛 | 深度烘焙麦芽，不透光 |

##### 闻香

- **麦芽香**：面包、饼干、焦糖、太妃糖、咖啡、巧克力
- **酒花香**：花香、柑橘、热带水果、松针、草本
- **酵母香**：香蕉、丁香、柑橘（小麦啤）；奶油、太妃糖（拉格）

##### 品味

- **苦度（IBU）**：国际苦度单位，淡啤 10-20，IPA 40-70，双 IPA 60-100+
- **甜度**：来自未发酵糖，帝国世涛和牛奶世涛偏甜
- **酒体**：从轻薄（淡拉格）到厚重（帝国世涛）
- **杀口感**：CO₂ 带来的刺激感，拉格通常更强

##### 余味

- 好的啤酒余味干净、持久，苦味和麦芽香在口中慢慢消散
- 劣质啤酒可能有酸败、纸板味、双乙酰（黄油味）等缺陷

#### 啤酒文化

##### 德国啤酒纯净法（Reinheitsgebot）

1516 年由巴伐利亚公爵威廉四世颁布，规定啤酒只能用三种原料酿造：大麦、酒花、水（当时酵母尚未被发现）。1892 年酵母被正式加入法规。这是世界上最古老的食品法规之一

##### 比利时修道院啤酒（Trappist）

由特拉普派修士在修道院内酿造的啤酒。全球仅有 14 家认证修道院酒厂，代表品牌：

- **Westvleteren（西弗莱特伦）**：全球最负盛名的修道院啤酒，12 号被誉为"世界最佳啤酒"
- **Chimay（智美）**：蓝帽、白帽、红帽三种风格
- **Rochefort（罗斯福）**：6 号、8 号、10 号，数字越大酒精度越高
- **Orval（奥沃尔）**：唯一使用干瓶发酵的修道院啤酒

##### 慕尼黑啤酒节（Oktoberfest）

每年 9 月中旬至 10 第一个周一举行，为期约 16-18 天。只允许使用特定风格的 Festbier（节庆啤酒），由慕尼黑六大酒厂供应：Paulaner、Hofbräu、Spaten、Löwenbräu、Hacker-Pschorr、Augustiner

#### 啤酒小知识

- 啤酒中的"原麦汁浓度"（°P / Plato）：指发酵前麦汁中可溶性物质的百分比，不是酒精度。12°P 表示麦汁中 12% 是可溶性物质
- 同一原麦汁浓度下，发酵度越高，酒精度越高，残糖越少，口感越干
- 啤酒的最佳饮用温度：小麦啤 4-7°C，IPA 7-10°C，世涛 12-16°C，兰比克 10-13°C
- 啤酒瓶颜色含义：棕色瓶防紫外线效果最好，绿色瓶次之，透明瓶最差。比利时很多啤酒用透明瓶，因为风味不依赖光敏化合物
