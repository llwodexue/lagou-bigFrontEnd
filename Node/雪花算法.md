## 雪花算法

雪花算法（Snowflake Algorithm）是由 Twitter 开发的一种分布式 ID 生成算法，旨在生成全局唯一且有序的 64 位整数 ID。这种算法特别适合在高并发、分布式系统中生成唯一 ID

> 雪花算法原生设计是为了生成 64 位的 ID，但如果需要生成 32 位的 ID，可以通过对原算法进行一些调整来实现

**雪花算法的 ID 结构**

```bash
# 64位结构
1 bit (符号位) | 41 bits (时间戳) | 10 bits (机器标识) | 12 bits (序列号)
# 32位结构
1 bit (符号位) | 10 bits (时间戳) | 5  bits (机器标识) | 16 bits (序列号)
```
