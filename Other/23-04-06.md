## 以客户为中心的经营增长

1. 需求？体验经济，客户到底需要什么

2. 客户？如何深度激活存量客户商业潜力

   理性人（熊大）、感性人（熊二）

   - 服务型（好的退票流程、解决各种投诉）
   - 体验型（奥利奥中间的果酱 -> 连接；口味改变，改变的果酱 -> 不需要改变自身产品。面饼 -> 企业面、客户面）

3. 卖点？行业玩法雷同，你如何吸引客户

   四个核心能力 -> 蝴蝶翅膀

4. 功能？产品服务功能堆砌，似乎做到尽头

- 需求（来自市场、领导） -> 产品 -> 卖点 -> 客户 -> 渠道

3. 价值甄别

   KANO 模型对需求的拆分

   - 期望属性 车能够尽快赶到

   - 无用属性 为乘客打开音乐
   - 反向属性 接单后给乘客致电
